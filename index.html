Research

Reinforcement Learning Research
2021 - Present

  • Enabled a constrained quadruped robot platform to overcome cheap hardware through artificial intelligence
  • Modeled a reinforcement learning problem of a constrained robot in simulation using OpenAI Gym & PyBullet
  • Applied TinyML techniques including Float16 quantization and graph freezing to reduce the ML model size
  • Transferred the Stable Baselines model to TFLite and deployed the ML model on a limited computing platform
Machine Learning Security Engineer
Summer 2022

  • Developed an artificial intelligence system to secure network traffic on board US Navy ships
  • Simulated US Navy ships navigating to waypoints while under cyber-attack using ROS, Gazebo, & OpenAI Gym
  • Trained a reinforcement learning agent to detect attacks on a US Navy ship’s network traffic and trigger alerts
  • Interpreted model by developing a dashboard in Flask to easily visualize network traffic and detected attacks
Mathematics Research (SPUR)
Summer 2020

  • Applied optimal control theory to study bifurcations in the construction of elastic Möbius bands using MATLAB
  • Computed the critical twist angle of a Möbius band using numerical methods such as the Runge-Kutta method
  • Discovered the emergence of a supercritical pitchfork bifurcation at the points of instability
  • Discovered that all branches emerging from bifurcations occurring before 180° lead to the same Möbius shape
Smart healthcare Research
Fall 2019 - Spring 2022

  • Applied machine learning techniques to improve Alzheimer’s patient-caregiver relationships
  • Led the development of a recommender system to provide adaptive recommendations to Alzheimer caregivers
  • Modeled the recommender system as a contextual multi-armed bandit reinforcement learning problem
  • Developed the recommender system using Python, SQL, XML RPC, AWS EC2, and HTML injection methods
  • Worked with healthcare professionals at OSU to deploy the RL and ML driven system in caregiver homes
